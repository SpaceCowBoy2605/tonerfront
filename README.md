# tonerfront

Frontend de la gestión de toners construido con React + Vite.

## Desarrollo

1. Instala dependencias: `npm install`
2. Servidor de desarrollo: `npm run dev` (http://localhost:5173)
3. Lint: `npm run lint`
4. Build producción: `npm run build` y `npm run preview`
5. Instalar Bootstrap: `npm install bootstrap`
6. instalar router: `npm install react-router-dom@6`, `npm ls react react-dom react-router-dom --all`
7. Cambiar a react 18: `npm ls react react-dom react-router-dom --all`,`npm install react@18 react-dom@18 --save-exact`
8. iconos de bootstrap: `npm i bootstrap-icons`


## Tecnologías

- React
- Vite
- ESLint
