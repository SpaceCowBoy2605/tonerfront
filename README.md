# tonerfront

Frontend de la gestión de toners construido con React + Vite.

## Desarrollo

1. Instala dependencias: `npm install`
2. Servidor de desarrollo: `npm run dev` (http://localhost:5173)
3. Lint: `npm run lint`
4. Build producción: `npm run build` y `npm run preview`

## Tecnologías

- React
- Vite
- ESLint
